# 策略健康度評估系統

量化交易策略績效評估工具，支援單一策略分析與多策略組合比較。

## 支援平台

- **XQ** — CSV 格式（Big5 / UTF-8 編碼）
- **MultiCharts** — XLSX 格式（交易明細匯出）
- **TradingView** — XLSX 格式（策略測試器匯出）

## 功能

### 單一策略評估
- 健康度評分 (100分制，S/A/B/C/D 等級)
- 核心指標：總淨利、勝率、獲利因子、期望值、賺賠比、最大回撤、恢復因子
- 進階指標：Sharpe Ratio、SQN、連勝/連敗、每K棒獲利
- 多空分析：做多/做空分別績效
- 圖表：權益曲線、獲利分佈、月度/年度損益
- 評分明細：8個維度拆解

### 組合策略評估（最多10個）
- 多策略並列比較總表（含不同交易區間）
- 時間對齊權益曲線
- 健康分數雷達圖
- 月度損益相關性矩陣

## 使用方式

### EXE（免安裝）
從 [Releases](../../releases) 下載 `策略健康度評估系統.exe`，雙擊即可執行。

### HTML（瀏覽器）
直接開啟 `策略評估系統.html`。

### 從原始碼執行
```bash
pip install pywebview
python app.py
```
